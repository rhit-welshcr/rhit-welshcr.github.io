KIC Mini Golf Senior Capstone

This project is a mini golf hole, intended to be played by prospective visiting students as well as anyone else on Rose-Hulman's campus. The hole includes a Plinko board and various electro-mechanical systems to allow for keeping score and to add a little extra fun and surprise to the experience. A website that showcases the main components of the project is currently being developed.
